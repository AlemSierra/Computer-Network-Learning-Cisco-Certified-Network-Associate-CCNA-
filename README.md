The purpose of this repository is to centralize, organize, and document all study material related to network learning under the Cisco Certified Network Associate (CCNA) training plan. Here you will find theoretical resources, labs, presentations, quizzes, and assessments from the three courses that make up the official learning path.

This repository aims to serve as a comprehensive and progressive guide that allows learners to understand everything from basic networking concepts to advanced topics such as security, automation, and enterprise architectures, ensuring structured and clear access to the content.
