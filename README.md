<p align="left">
  
El propósito de este repositorio es centralizar, organizar y documentar todo el material de estudio relacionado con el aprendizaje de redes bajo el plan de formación de Cisco Certified Network Associate (CCNA). Aquí se recopilan recursos teóricos, laboratorios, presentaciones, quizzes y evaluaciones de los tres cursos que componen la ruta oficial.

Este repositorio busca servir como una guía integral y progresiva que permita comprender desde los conceptos básicos de redes hasta temas avanzados como seguridad, automatización y arquitecturas empresariales, garantizando un acceso estructurado y claro al contenido.

</p>
